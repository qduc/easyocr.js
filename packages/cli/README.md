# @qduc/easyocr-cli

Command-line interface for [easyocr.js](https://github.com/qduc/easyocr.js).

## Installation

```bash
npm install -g @qduc/easyocr-cli
```

## Usage

```bash
easyocr path/to/image.png --langs en,ch_sim
```

See the [main repository README](https://github.com/qduc/easyocr.js#readme) for more details.
